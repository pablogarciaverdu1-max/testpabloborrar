Slide 02
Enrutamiento Avanzado y Lógica de Optimización Continua
El sistema de niveles inteligente garantiza que el BDR correcto contacte la cuenta correcta en el momento oportuno.
Nivel 1
Cuentas Estratégicas
Criterios
  • Encaje perfecto con el ICP
  • Alto potencial de ARR
  • Marcas estratégicas
Enrutamiento
  • Asignado a BDR senior
  • Investigación manual requerida
  • Secuencia de outreach personalizada
Nivel 2
Cuentas de Crecimiento
Criterios
  • Buen encaje con el ICP
  • Valor de oportunidad medio
Enrutamiento
  • Secuencias semi-personalizadas
  • Mensajería inicial generada por IA
Nivel 3
Cuentas de Volumen
Criterios
  • Bajo encaje con el ICP
  • Encaje poco claro
Enrutamiento
  • Campañas outbound totalmente automatizadas
Lógica de Enrutamiento Dinámico
SI
  Industria = Retail / Moda / Artículos Deportivos
  Y Cargo contiene:
    CEO  ·  Category Manager  ·  Director de Ventas

ENTONCES Asignar Puntuación de Prioridad

  Puntuación Alta    → Nivel 1
  Puntuación Media → Nivel 2
  Puntuación Baja   → Nivel 3
Bucle de Optimización Continua
Métricas de Revisión Semanal
  • Tasa de respuesta
  • Tasa de reuniones cerradas
  • Tasa de SQL
  • Tasa de creación de oportunidades
Acciones de Optimización
  • Ajustar umbrales de puntuación
  • Actualizar filtros de ICP
  • Promover segmentos de alto rendimiento
  • Eliminar segmentos de bajo rendimiento
Resultado
Los BDRs dedican su tiempo solo a las cuentas con mayor probabilidad de conversión.
